# memory 🚧 WIP

> 跨会话记忆工具集——把"上下文"沉淀下来，让 Claude 在新会话里也能找回上次的状态。

> ⚠️ **当前为占位骨架**，暂未在根 `marketplace.json` 注册。第一个 skill 落地后会一并发布并升到 `0.1.0`。

## 设计目标

- **长期记忆持久化**：把项目级 / 用户级的关键事实结构化保存，而不是塞 CLAUDE.md
- **跨会话回溯**：在新会话开始时自动检索"上次到哪了 / 上次的决策是什么"
- **可检索 / 可裁剪**：记忆不是越多越好，提供查询、合并、过期清理能力
- **本地优先**：默认数据落本地，不上传第三方

## 计划中的 skills

| Skill | 状态 | 说明 |
| --- | --- | --- |
| `memory-store` | 📝 计划 | 结构化记录事实 / 决策 / 偏好，带类型与索引 |
| `memory-recall` | 📝 计划 | 按主题 / 时间 / 关键词检索历史记忆 |
| `memory-prune` | 📝 计划 | 检测陈旧记忆并提示更新或清理 |
| `session-resume` | 📝 计划 | 新会话开始时自动加载相关上下文 |

> 这些只是设想，最终落地时会按"先一个能跑起来再说"的节奏推进。
